# 🎮 Gaming Landing Page - Angular Project

Este proyecto es una Landing Page moderna para una plataforma de videojuegos, desarrollada con **Angular**. Cuenta con una interfaz dinámica, un catálogo de juegos interactivo y una sección principal (Hero) con transiciones automáticas.

## 🚀 Características Principales

* **Hero Dinámico:** Slider automático que cambia de juego destacado cada 5 segundos.
* **Catálogo Interactivo:** Sección de juegos con etiquetas de estado (Nuevo, Oferta, Más Vendido).
* **Diseño Responsivo:** Adaptado para ofrecer una experiencia visual inmersiva.
* **Optimización de Recursos:** Gestión de activos locales mediante la configuración de rutas en `angular.json`.

## 🛠️ Tecnologías Utilizadas

* **Framework:** [Angular](https://angular.io/) (Componentes Standalone)
* **Lenguaje:** TypeScript
* **Estilos:** CSS3 con variables personalizadas para temática gaming.
* **Iconografía:** Font Awesome / Google Material Icons.
